# Reverse polish notation to polish notation converter

## Installation

```bash
    git clone https://github.com/Beaxhem/architecture-lab-2
```

## Usage

```bash
    go run cmd/example/main.go
    2 2 *
    * 2 2 // Output
```

### Available flags

    -h, --help:
        Show this help message.
    -e, 
        Pass expression directly.
    -f:
        Input file.
    -o:
        Output file.
